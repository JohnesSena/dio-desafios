< mark>

O elemento HMTL < mark> **representa um trecho de destaque em um texto, por exemplo, sequência de texto marcado como referência, devido à sua relevância em um contexto particular. Por Exemplo, pode ser utilizado em uma página que mostra os resultados de uma busca onde todas as instâncias da palavra pesquisadas receberam destaque.

< small>

Eu sei o que você está pensando agora: “ < small> é deprecated, ninguém usa mais desde que o CSS passou a definir o tamanho das fontes dos elementos ”. Tirando a parte do deprecated, você está até certo. Mas isso foi antes do HTML 5. Com o advento do HTML 5, o < small> deixou de ser aquela tag sem graça que apenas reduzia o tamanho da fonte para ganhar um valor semântico. Agora, a tag simboliza elementos que normalmente seriam escritos em letra miúda fora da web, como avisos, regras, restrições legais, direitos de uso etc.

< sub>

O < sub> o elemento deve ser utilizado apenas por razões tipográficas — ou seja, para alterar a posição do texto de modo a cumprir as convenções ou normas tipográficas, e não apenas para fins de apresentação ou aparência. Por exemplo, usando < sub> para estilizar o nome de uma empresa que usa linhas de base alteradas em suas marca de palavra não seria apropriado; em vez disso, o CSS deve ser usado. Por exemplo, você pode usar o vertical-align propriedade com uma declaração como vertical-align: sub ou, para controlar com mais precisão a mudança de linha de base, vertical-align: -25%. Casos de uso apropriados para < sub> inclua (, mas não esteja necessariamente limitado a ): Marcando números de notas de rodapé. Veja Números de nota de rodapé por exemplo. Marcando o subscrito em números variáveis matemáticos ( embora você também considere usar um MathML fórmula para este ). Veja Subscritos variáveis. Denotando o número de átomos de um determinado elemento dentro de uma fórmula química (como o melhor amigo de cada desenvolvedor, C H 10 N 4 O 2 , também conhecido como "cafeína").

< sup>

O < sup> o elemento só deve ser utilizado por razões tipográficas—ou seja, para alterar a posição do texto de modo a cumprir as convenções ou normas tipográficas, e não apenas para fins de apresentação ou aparência. Por exemplo, para estilizar o marca-palavra de um negócio ou produto que usa uma linha de base elevada deve ser feito usando CSS (provavelmente vertical-align) em vez de < sup>. Isso seria feito usando, por exemplo, vertical-align: super ou, para mudar a linha de base para cima 50%, vertical-align: 50%. Casos de uso adequados para < sup> incluir (mas não estão necessariamente limitados a): Expor expoentes, como "x3". Pode valer a pena considerar o uso de MathML para estes, especialmente em casos mais complexos. Ver Exponentes sob Exemplos abaixo. Exibindo letras superiores, que é usado em alguns idiomas ao renderizar certas abreviações. Por exemplo, em francês, a palavra "mademoiselle" pode ser abreviada "Mlle"; este é um caso de uso aceitável. Ver Letras superiores por exemplo. Representando números ordinais, como "4th" em vez de "quarto." Ver Números ordinais por exemplo.
